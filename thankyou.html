Machine Learning for Data Science

  1. Tableau:
    • Introduction to Tableau and its interface
    • Creating charts, graphs, and maps in Tableau
    • Connecting to different data sources
    • Working with calculated fields and parameters
    • Using Tableau for data storytelling and presentation
  2. Probability and Random Variable:
    • Basic probability concepts and notation
    • Conditional probability and Bayes' theorem
    • Discrete and continuous random variables
    • Probability distributions and their properties
    • Joint and marginal probability distributions
  3. Distribution:
    • Normal distribution and its properties
    • Binomial distribution and its properties
    • Poisson distribution and its properties
    • Exponential distribution and its properties
    • Chi-squared distribution and its properties
  4. Hypothesis Testing:
    • Hypothesis testing process and its components
    • One-sample and two-sample t-tests
    • ANOVA (Analysis of Variance) and F-tests
    • Non-parametric hypothesis tests
    • Power and sample size calculations
  5. Feature Engineering:
    • Feature selection techniques
    • Data normalization and scaling
    • Handling missing data
    • Encoding categorical variables
    • Feature extraction techniques
  6. EDA (Exploratory Data Analysis):
    • Data cleaning and preprocessing
    • Data visualization techniques
    • Descriptive statistics
    • Outlier detection and removal
    • Data transformation and normalization
  7. Logistic Regression:
    • Binary and multinomial logistic regression
    • Odds ratio and its interpretation
    • Model fitting and diagnostics
    • Regularized logistic regression techniques
    • Applications of logistic regression
  8. Model Deployment:
    • Model deployment strategies and considerations
    • Web deployment with Streamlit
    • Model performance evaluation
  9. Unsupervised Machine Learning:
    • Clustering algorithms (K-means, Hierarchical clustering)
    • Dimensionality reduction techniques (PCA, t-SNE)
    • Association rule mining and market basket analysis
    • Recommender systems and collaborative filtering
    • Anomaly detection techniques
  10. Decision Trees:
    • Decision tree building and splitting criteria
    • Handling missing values and categorical variables
    • Decision tree pruning and regularization
    • Ensemble techniques (Random Forests)
    • Model interpretation and feature importance
  11. Supervised Machine Learning:
    • Classification algorithms (Logistic Regression, Naive Bayes)
    • Regularization techniques (Ridge, Lasso, Elastic Net)
    • Support Vector Machines (SVM) for classification and regression
    • Gradient Boosting and AdaBoost
    • Neural networks and deep learning
  12. Case Study:
    • Real-world data science project examples
    • End-to-end data science project implementation
    • Best practices and project management strategies
    • Presentation and communication skills for data science professionals
    • Latest trends and developments in data science and machine learning.

    • Convolutional neural networks (CNN)

  2. Text Mining:
    • Preprocessing and cleaning text data
    • Text feature extraction techniques
    • Text classification and clustering
    • Topic modeling techniques
    • Sentiment analysis
